Class TransformingStream<TInput, TOutput>

The TransformingStream is a wrapper class that abstracts the grpc.ClientReadableStream interface away from the user and transforms the input from the grpc format to Node objects.

This is used when the Library sensor is providing a stream of events, instead of a one-time response.
